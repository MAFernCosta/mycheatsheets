# Basic markdown sintax

## Headings

# Heading 1

```markdown
# Heading 1
```

## Heading 2

```markdown
## Heading 2
```

### Heading 3

```markdown
### Heading 3
```


## Text Formatting

### Bold

**Bold text**

```markdown
**Bold text**
```

### Italic

*Italic text*

```markdown
*Italic text*
```

### Bold and Italic

***Bold + Italic***

```markdown
***Italic text***
```

## Blockquote

    > Blockquote

## Lists

Unordered:

    - Item
    - Item

Ordered:

    1. Item
    2. Item

## Code

Inline:

    `Inline code`

Block:

``` bash
echo "hello"
```

## Links & Images

    [Link text](https://example.com)
    ![Alt text](image-url.jpg)

## Horizontal Rule

    ---

## Table

    | Column A | Column B |
    |----------|----------|
    | Item 1   | Item 2   |
